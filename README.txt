PARA CRIAR ARQUIVOS NOVOS NO JOGO É PRECISO FAZER O SEGUINTE:

1° DEFINIR O MODO DE JOGO (1 APENAS TRADUÇAO, 0 COGNIÇAO SIMBOLICA E TRADUÇAO, A COGNIÇAO SIMBOLICA DEVE SER USADA PARA LINGUAS QUE EXCEDEM OS CARACTERES ASCII)

2° COLOCAR O NUMERO MAXIMO DE POSSIBILIDADES DO JOGO (O MAXIMO DE PALAVRAS, SABENDO QUE O 0 CONTA COMO 1)

3° DEFINIR A IMAGEM A SER USADA COMO TUTORIAL, CASO NAO TENHA, COLOCAR UM - (TRAÇO) NO LOCAL -- ESSA IMAGEM DEVE ESTAR NA PASTA Imagens

4° DEFINIR O ARQUIVO DE FONTE (.TTF) QUE SEU JOGO USARÁ -- ESSA FONTE DEVERÁ ESTAR NA PASTA Fontes

5° CADA LINHA DEVE SER INICIADA POR UM NUMERO (COMEÇANDO EM 0, NAO PODE SER DECIMAL(COM VIRGULA) NEM NEGATIVO). EM SEQUENCIA DEVE SER COLOCADO, SEPARADO POR ESPAÇOS, A PALAVRA DE EXIBIÇAO, ESCRITA E TRADUZIDA NO FINAL QUEBRE A LINHA.

6° NO FINAL DA ULTIMA PALAVRA DE UM ENTER PARA FICAR COM UMA LINHA VAZIA NO FINAL.
